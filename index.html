This club is all about the Catan Board Game

What is Catan?

Catan is a multiplayer board game that was first published in 1995 where each player controls a settlment. Players work to develop their settelments and accumulate victory points. Typically the first person to 10 points wins. View the rules page for more details on how the game works.
